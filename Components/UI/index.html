UI Components

The goal of this page is to present and reference different reusable UI components in the project, and detail their usage.

MenuBox

Description

This component is a basic UI component for displaying a floating box, and displaying it only when a condition is met.

Props

activate
bool
onClose
func
callOnChange
func
watchChange
array

Example

The following example shows how you can use a MenuBox to display something when clicking on a button. Every element that is a child of MenuBoxContent is displaying in the pop up box, the rest is displayed as normal.

I'm the content of the box ! 

() => {
  // MenuBoxExemple
  const [activated, setActivated] = useState(false)
  return (
    <MenuBox
      activate={activated}
      style={{ textAlign: 'center' }}
      onClose={() => setActivated(false)}
    >
      <button
        onClick={() => {
          setActivated(!activated)
        }}
        style={buttonStyle}
      >
        Click me to open a box
      </button>
      <MenuBoxContent>
        <p style={paragraphStyle}> I'm the content of the box ! </p>
      </MenuBoxContent>
    </MenuBox>
  )
}

This exemple uses a React state to keep track of the state of the MenuBox (opened or closed).

Tooltip

Description

This component is similar to the MenuBox component in that it displays a floating box on top of elements.

Props

This component has no useful prop.

Example

The following example shows how you would use the Tooltip component, as mentioned earlier it's similar to the MenuBox component. When hovering the Tooltip component, the children of the TooltipContent component will be displayed in a box.

Hover me for a tooltip

Here's a tooltip 

() => {
  // TooltipExemple
  return (
    <Tooltip>
      <div style={divStyle}> Hover me for a tooltip </div>
      <TooltipContent>
        <p style={paragraphStyle}> Here's a tooltip </p>
      </TooltipContent>
    </Tooltip>
  )
}
